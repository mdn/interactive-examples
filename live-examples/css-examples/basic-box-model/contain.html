  • With size, the element will not grow or shrink to fit its content. This element has a width and min-height so it doesn't collapse to 0x0px.
  • With layout, the fixed box uses the containing element as a layout context.
  • With paint, elements that overflow the bounding box of the containing element are not painted.
  • With strict, all types of containment are applied.
